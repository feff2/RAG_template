Организатора  для общих  закупок  223  отображается  строка  «Предоставлять  ЭП»  с
возможностью выбора значения из выпадающего списка:
•  На всех этапах (установлено по умолчанию) - система запрашивает ЭП на
всех этапах процедуры;
•  При заключении договора - система не запрашивает ЭП на следующих
этапах:
•  Создание извещения;
•  Редактирование извещения;
•  Рассмотрение заявок;
•  Рассмотрение 1-х частей;
•  Рассмотрение 2-х частей;
•  Рассмотрение заявок без ЦП;
•  Вскрытие конвертов;
•  При публикации/отклонении запроса на разъяснение положений
документации;
•  При публикации протоколов;
•  Подведение итогов квал. Отбора;
•  Протокол очной переторжки;
•  Вскрытие конвертов по заочной переторжке;
•  Протокол подачи ценовых предложений аукциона;
•  Подведение итогов.
Система запрашивает ЭП на следующих этапах:
•  Подписание договора в электронной форме;
•  Публикация протокола уклонения и отказа;
•  Заключение дополнительных соглашений;
•  Публикация проекта договора;
•  Отправка сведений о договоре в ЕИС;
•  Отправка сведений о расторжении, исполнении договора, а также о
договорах с субподрядчиками.
•  Ни  на  каком  этапе  -  система  не  запрашивает  ЭП  ни  на  одном  этапе
процедуры.

Преднастроить  условия  проведения  закупок  без  использования  ЭП
Организатор  может  в  настройках  организации  (подробнее  в  п.п  4.2.1
текущего документа).
В случае, если Оператор отключает настройку, то для новых процедур ЭП
будет запрашиваться на всех этапах проведения процедуры. Процедуры,
которые уже были опубликованы с включенной настройкой, работают без
ЭП на соответствующих настройкам этапах.
Параметр извещения «Предоставлять ЭП» можно отредактировать только
в черновике создаваемого извещения.
В  черновик  извещения  автоматически  подтягиваются  аккредитационные
сведения  Организатора  из  личного  кабинета  в  Росэлторг.ID.  Пока  извещение  не
опубликовано, у Организатора есть возможность актуализировать аккредитационные
сведения своей компании в случае, если данные изменились. Для этого Организатору
необходимо  актуализировать  данные  в  личном  кабинете  в  Росэлторг.ID,  далее  в
черновике  извещения  на  вкладке  «Общая  информация»  в  блоке  «Сведения  об
организаторе» выбрать действие «Обновить актуальность данных» (Рис. 440).
