альтернативных предложений
При
активации
параметра
«Возможность
подачи
альтернативных
предложений»  в  настройках  организации,  данный  параметр  будет  отображаться  по
умолчанию на форме создания черновика извещения для процедур:
•  конкурс;
•
запрос предложений;
•  конкурентные переговоры.
Процесс  создания  извещения  с  возможностью  подачи  альтернативных
предложений  не  отличается  от  стандартного  процесса  создания  извещения.  Более
подробно процесс создания извещения представлен в Руководстве пользователя.
Для  проведения  процедуры  с  возможность  подачи  альтернативных
предложений  Организатор  на  вкладке  «Общая  информация»  может  установить
параметр  «Возможность  подачи  альтернативных  предложений»,  если  данный
параметр не был активирован в настройках организации (Рис. 961).
Подача  альтернативных  предложений  выполняется  только  в  период
проведения этапа «Прием заявок»
