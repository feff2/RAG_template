Рис. 476 Блок «Параметры аукциона»
В  поле  «Длительность  шага  аукциона»  по  умолчанию  указано  10  минут,
Организатор  может  указать  другую  длительность,  введя  значение  вручную  либо  с
помощью стрелочек.
Длительность  шага  аукциона  не  может  быть  менее  1  минуты  и  более  60
минут.  В  качестве  значения  длительности  шага  аукциона  может  быть
указано только целочисленное значение.
При  установке  признака  «Указать  логику  снижения  предложения  о  цене
договора» открываются дополнительные поля для заполнения (Рис. 477).
По  умолчанию  в  поле  «Снижение  цены  от  НМЦ»  указано  значение  «С  НДС».
Данное  поле  обязательно  для  заполнения.  При  заполнении  данного  параметра
проверка при подаче ценового предложения с учетом шага аукциона в торгах будет
осуществляться  по  полю  ценового  предложения  (с  НДС  /  без  НДС)  на  основании
указанного Заказчиком признака «Снижение цены от НМЦ» (с НДС / без НДС).
В поле «Ставка НДС» Организатор выбирает значение из выпадающего списка
(Рис. 478). Данное поле обязательно для заполнения.
Значение  в  поле  «НМЦ  с  НДС»  заполняется  автоматически,  равно  значению
поля «Начальная максимальная цена» с учетом НДС в блоке «Сведения о начальной
цене  договора».  С  учетом  значения  в  поле  «Ставка  НДС»  также  автоматически
заполняется поле «НМЦ без НДС» (Рис. 479).
