Рис. 356. Кнопка «Добавить позицию»
Откроется  окно  «Товар,  работа,  услуга»  для  ввода  сведений  о  позиции.  Все
поля,  обязательные  для  заполнения,  отмечены  символом  «*».  Указываются
следующие сведения (Рис. 357):
•
«Наименование  ТРУ»  —  текстовое  поле,  должно  содержать  наименование
позиции;
•
«Тип объекта закупки» — доступен выбор из выпадающего списка:
«Товар»;
«Работа»;
«Услуга».
•
•
«Категория» — код позиции по справочнику ОКПД2;
«Единица  измерения»  —  единица  измерения  позиции  по  классификатору
ОКЕИ;
•
«Количество  (объем)  ТРУ»  —  указывается  число  —  точное  количество
(объем).  В  случае  если  нет  возможности  определить  количество  (объем)
ТРУ, необходимо активировать соответствующий признак;
•
«Реестр» — реестр, в котором зарегистрирована позиция ТРУ, необходимо
выбрать из выпадающего списка;
•
«Реестровый  номер»  —  реестровый  номер  позиции  ТРУ  в  выбранном
реестре;
•
«Страна  происхождения»  —  доступен  поиск  при  вводе  названия  в  поле,
также  возможен  выбор  из  выпадающего  списка.  Поле  может  содержать
несколько значений — стран;
•
«Цена  за  единицу»  —  числовое  значение,  доступно  округление  до  двух
знаков после запятой;
Корпоративные закупки
и закупки субъектов 223-ФЗ (включая МСП)
Стр. 260 из 758
•
•
«Валюта» — доступен выбор из выпадающего списка;
«Всего» — поле заполняется автоматически при указании значений в поле
«Количество (объем) ТРУ» и «Цена за единицу», не редактируется;
•
«Тип  финансового  обеспечения»  —  доступен  выбор  из  выпадающего
списка, обязательно для заполнения.
